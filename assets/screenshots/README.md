# Screenshots

The main README uses the vector graphics in `../banner.svg` and
`../photon-paths.svg` (which always render crisply).

Want to add **real screenshots** of your own session? It takes 10 seconds:

1. Open the app and set up a view you like (try a preset + an inclination).
2. Press **`P`** (or click the **⌖** button) — it saves a PNG to your Downloads.
3. Drop the PNGs in this folder and reference them in the README, e.g.:

   ```markdown
   ![Sgr A* face-on](assets/screenshots/hero.png)
   ![Photon paths](assets/screenshots/photon-paths.png)
   ```

Suggested shots: a face-on disk + photon ring, an edge-on "Interstellar" view,
the photon-path tracer (Geometry → Photon paths, inclination → 0), and one with
the control panel + live readouts visible.
